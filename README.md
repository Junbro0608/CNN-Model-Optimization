# CIFAR-10 이미지 분류를 위한 CNN 모델 최적화 연구
> **인공지능기초 기말 프로젝트** > **시스템반도체공학과 이준형**

[cite_start]본 프로젝트는 CIFAR-10 데이터셋을 활용하여 이미지 분류 성능을 높이기 위한 신경망 구조를 설계하고, 다양한 가설 검증을 통해 최적의 모델을 구축하는 과정을 담고 있습니다

---

## 1. 프로젝트 개요
- **데이터셋**: CIFAR-10 (10개 클래스, 60,000개 32x32 컬러 이미지)
- **목표**: 50,000개의 훈련 이미지를 학습시켜 테스트 이미지에 대한 정확도(Accuracy) 극대화
- **핵심 요소**: CNN vs DNN 구조 비교, ELU 활성화 함수 도입, He 가중치 초기화 적용

---

## 2. 주요 가설 및 검증

### 2.1 CNN vs DNN (공간 정보 유지)
- **가설**: DNN은 이미지 데이터를 1차원으로 변환하며 공간적 정보를 손실하지만, CNN은 합성곱(Convolution) 연산을 통해 픽셀 간 연관성을 유지하므로 성능이 더 뛰어날 것이다
- **결과**: 동일 조건에서 CNN 기반 모델이 DNN보다 현저히 높은 성능을 기록하여 가설을 증명하였다

### 2.2 활성화 함수 (ELU vs ReLU)
- **가설**: ReLU의 음수 데이터 기울기 소실(Dying ReLU) 문제를 보완하기 위해 ELU를 사용하면 성능이 향상될 것이다
- **결과**: 실험 결과, 모든 네트워크에서 ELU를 적용했을 때 더 높은 정확도를 도출하였다

## 3. 실험 결과 요약 (Result Analysis)

[cite_start]학습 횟수와 네트워크 구조에 따른 성능 비교 데이터입니다[cite: 45, 57].

| 네트워크 종류     | 학습 횟수  | 활성화 함수 | Test Acc (%) |

## 4. 최종 네트워크 설계 (Model Architecture)

가설 검증을 통해 설계된 최적의 모델 구조입니다[cite: 67, 73, 82].

- **구조**: `Conv + MaxPool` → `Conv + MaxPool` → `Conv` → `Affine` (3계층 연산)[cite: 68, 69].
- **활성화 함수**: **ELU** (모든 활성화 계층에 적용)[cite: 82].
- **가중치 초기화**: **He 초기화** (기울기 손실 방지 및 균일한 분포 형성).
- **최종 성능**: 테스트 정확도 **66.22%** 달성.

---

## 5. 결론
실험을 통해 이미지 처리에는 공간 정보를 유지하는 CNN이 필수적이며, ELU 활성화 함수와 He 가중치 초기화를 결합했을 때 학습의 안정성과 정확도가 가장 높다는 것을 확인하였습니다
